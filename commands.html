Some common commands

  • ls - List directory contents.
    Usage: ls or ls directory_name.
  • cd - Change directory.
    Usage: cd directory_name.
  • pwd - Print working directory.
    Usage: pwd.
  • mkdir - Make directory.
    Usage: mkdir directory_name.
  • rm - Remove files or directories.
    Usage: rm file_name or rm -r directory_name.
  • cp - Copy files or directories.
    Usage: cp source_file destination_file or cp -r source_directory destination_directory.
  • mv - Move or rename files or directories.
    Usage: mv old_file new_file or mv old_directory new_directory.
  • cat - Display file contents.
    Usage: cat file_name.
  • grep - Search for a string in a file.
    Usage: grep "search_string" file_name.
  • chmod - Change file permissions.
    Usage: chmod octal_permissions file_name.
  • sudo - Execute a command with elevated privileges.
    Usage: sudo command.
  • ssh - Connect to a remote server.
    Usage: ssh username@server_ip.
  • scp - Copy files between a local and a remote machine.
    Usage: scp source_file username@server_ip:/destination_directory or scp username@server_ip:/source_directory/destination_file
  • tar - Archive files into a single file.
    Usage: tar -cvf archive_name.tar file_name or tar -xvf archive_name.tar.
  • top - Display system resource usage.
    Usage: top.
  • df - Display disk space usage.
    Usage: df.
  • du - Display disk usage of files and directories.
    Usage: du.
  • ping - Test network connectivity.
    Usage: ping server_ip.
  • traceroute - Display the route taken by network packets.
    Usage: traceroute server_ip.
  • ifconfig - Display network interface configuration.
    Usage: ifconfig
